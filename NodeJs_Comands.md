# NPM

### package.json
```bash
$ npm install                # Instala pacotes e bibliotecas - cria pasta node_modules
$ npm uninstall              # Remove pacotes não utilizados - exclui de node_modules
$ npm update                 # Atualiza versões de pacotes
$ npm init                   # Inicializa o npm no projeto
$ npm start                  # Executa a aplicação conforme o script start
$ npm run                    # Executa qualquer script definido no package.json

$ npm outdated               # verificar dependências que possuem atualização
$ npx npm-check-updates -u   # atualizar dependências para versões mais recentes
```
